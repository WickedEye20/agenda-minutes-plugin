== Description ==

The Agenda Minutes Plugin is a powerful tool for managing and organizing your meeting agendas and minutes directly within your WordPress site. Streamline your meeting documentation and make collaboration easier with this user-friendly plugin.

= Key Features =

- Create and manage meeting agendas.
- Document and store meeting minutes.
- Organize and categorize meetings.
- Customizable templates for agendas and minutes.

== Installation ==

1. Upload the `agenda-minutes-plugin` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Usage ==

1. Create a new meeting by navigating to the "Agenda/Minutes" menu in your WordPress dashboard.
2. Add agenda items and meeting details.
3. During the meeting, use the plugin to take and save minutes.
4. Customize your meeting agenda and minutes using templates.
5. Collaborate with team members to ensure accurate documentation.

== Frequently Asked Questions ==

= Do I need any technical skills to use this plugin? =

No, the plugin is designed with a user-friendly interface, making it accessible for users of all technical levels.

= Can I customize the agenda and minutes templates? =

Yes, you can easily customize the templates to match your organization's needs and branding.
